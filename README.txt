Program modyfikuje plik Przyklad.exe poprzez podmienienie funkcji sin() na cos().

Aby zauważyć działanie modyfikacji, należy najpierw wkleić oryginalny plik Przyklad.exe do projektu Modyfikacja_Pliku_PE, a następnie uruchomić Modyfikacja_Pliku_PE.exe.

Po uruchomieniu zmodyfikowanego Przyklad.exe można zauważyć zmienione działanie aplikacji.

Program nie ma zaimplementowanego sprawdzania poprawności danych, więc należy uważać aby podać mu prawidłowy plik .exe.
Przyklad.exe można skopiować z projektu aplikacji Przyklad.
